Ceci est la documentation de l’API REST pour le TP5. Ce service permet de manipuler un répertoire d’utilisateurs. Ainsi on peut ajouter, modifier ou supprimer un utilisateur mais aussi lister tout les utilisateurs présents en BDD ou un seul.

user

Opérations sur les utilisateurs

POST/user
Ajoute un nouvel utilisateur
GET/user
Affiche tous les users en base de donnée.
GET/user/{id}
Get user by id
PUT/user/{id}
Update user by id
DELETE/user/{id}
Delete user by id
GET/user/find
Get users filtered by first name, last name and/or id
